Calculadora de Triagens Neonatais

Uma ferramenta para cálculo automático de datas para exames de triagem neonatal

Sobre o Projeto

A Calculadora de Triagens Neonatais é uma aplicação web desenvolvida para auxiliar profissionais de saúde a determinar as datas ideais para triagens neonatais com base na data de nascimento, peso e idade gestacional do recém-nascido.

Funcionalidades

  • Cálculo automático de datas para exames de triagem neonatal:
    • USG de Crânio (ultrassom)
    • Fundo de Olho (retinopatia da prematuridade)
    • Doença Metabólica Óssea
    • Triagem Auditiva
  • Agendamento de USG apenas para terças e sextas-feiras
  • Avaliação de fatores de risco para HPIV
  • Diferentes protocolos de acompanhamento baseados nos resultados do USG
  • Cálculo de datas para testes de doença metabólica a cada 21 dias

Status do Projeto
